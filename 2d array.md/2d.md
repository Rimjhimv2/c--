Summary Notes: 2D Arrays & Row Major Mapping
1. 2D Array Storage in Memory

A 2D array (matrix) is stored in linear memory.

Elements are stored row by row (this is called Row Major Order).

Example:
Matrix (4 × 3)

00  01  02  
10  11  12  
20  21  22  
30  31  32


Stored in memory as:
00, 01, 02, 10, 11, 12, 20, 21, 22, 30, 31, 32

3. Finding Row and Column from Linear Index

If you already know the linear index (say Index) and total columns (col):

Row index = Index / col (integer division)

Column index = Index % col (remainder)

4. Reverse Mapping (from Index back to Row & Column)

If you already know the Index:

Row index = Index / totalCols

Col index = Index % totalCols

✅ Key Points to Remember:

Row major means row by row storage.

Formula links row/col position to linear memory.

Always multiply with element size when calculating real addresses.
